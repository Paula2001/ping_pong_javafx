# ping pong game 


### prerequisites : 

- Java 18.0

### important commands : 

- java -jar ./game.jar
 
### controllers

- "up arrow" : to move player one up 
- "down arrow" : to move player one down 
- "w" : to move player two up 
- "s" : to move player two down 
- "r" : to resume the game
- "p" : to pause the game
- when player score a game press "r"
